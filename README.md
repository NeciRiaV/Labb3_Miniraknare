5.	Tryck på enter för att komma tillbaka till menyn.

7.	Tryck enter.

9.	Tryck enter.
10.	Resultatet borde skriva ut ”Fel! Ogiltig räknesätt vald! Vänligen välj giltigt räknesätt!”.
